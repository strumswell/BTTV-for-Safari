# BTTV for Safari

This unofficial Safari exention adds support for BTTV and FFZ emotes on Twitch. The extension simply injects the BTTV and FFZ script from the official BTTV and FFZ CDN. With the newly added FFZ support, you can also enable add-ons like 7TV emotes or Chatterino badges.

### Download: [![Github All Releases](https://img.shields.io/github/downloads/strumswell/BTTV-for-Safari/total.svg)](https://github.com/strumswell/BTTV-for-Safari/releases/download/v1.1/BTTV.for.Safari.zip)

## Installation

1. Download the app
2. Drag into Applications folder
3. Double click (or right click > Open)
4. Click "Quit and Open Safari Extensions Preferences..."
5. Set checkmark for "BTTV for Safari"
6. Click on "BTTV for Safari", click "Edit webpages". Select "Allow" for twitch.tv **OR** click on `D:` emote in Safari task bar and allow always access.

Everything works, if the `D:` emote in your Safari task bar shines blue when on twitch.tv. Otherwise, click on it and check allow.
